MySQL export to csv

#dev #howto #mysql

To export a mysql results in CSV format, use the syntax below :

SELECT field1, field2, ...
FROM table
WHERE condition
INTO OUTFILE '/tmp/toto.csv'
FIELDS TERMINATED BY ';'
ENCLOSED BY '"'
LINES TERMINATED BY '\n' ;
