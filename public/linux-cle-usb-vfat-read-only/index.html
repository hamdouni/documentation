Comment réparer une clé usb vfat qui passe en read-only sous Linux ?

S’il n’est plus possible d’utiliser une clé usb sous linux avec le système qui bascule en read-only, et qu’on se retrouve avec ce genre de lignes dans le syslog :

FAT: Filesystem panic (dev sdb1)

fat_get_cluster: invalid cluster chain (i_pos 0)

un petit

dosfsck -a /dev/sdb1

et ça repart. La commande répare le file system et récupére les fichiers.

 

Si en plus, on souhaite formater la clé en lui donnant un label, c’est simple :

 mkfs.vfat -n ‘Votre_label_ici’ -I /dev/sdb1
