Let’s groove !

#Groovy #Java

Je me suis mis à Groovy. Non, non, rien à voir avec la chanson d’Earth Wind and Fire (aïe, je me fais vieux). Je parle du langage de programmation qui s’appuie sur la machine virtuelle java (la fameuse jvm) mais en proposant la souplesse des langages dynamiques.

Bref, pour me forcer à l’utiliser me motiver à l’apprendre, je code des minis applications qui pourront servir un jour : je me base sur l’idée d’Unix, plein de petites applis qui font qu’une chose mais qui le font bien. Et là, j’ai commencé avec un afficheur de code barre (tiens, tiens, mais pourquoi donc ?)

On entre un numéro de code barre sur 13 chiffres et l’appli web affiche l’image. Rien d’extraordinaire mais ça m’a obligé à traiter tous les aspects : de l’IDE jusqu’au déploiement. Et donc, l’appli tourne en live sur ce serveur :  http://brahim.hamdouni.com:8080/ean13toimg/

Allez, au suivant : un script qui va analyser un fichier log, analyser le contenu pour sortir des données pertinentes et l’enrichir avec des données depuis une base MySql. Ça devrait m’occuper…
